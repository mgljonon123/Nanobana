# Stay — House & Apartment Rentals

A simple e-commerce-style site for browsing and filtering house and apartment rentals.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## What’s included

- **Home** — Hero, search bar, and featured rentals
- **Rentals** — Full listing grid with filters (city, type, price, bedrooms)
- **Property detail** — Single listing with gallery, description, amenities, and “Request to rent” CTA

Data is mock (see `src/data/listings.ts`). You can plug in a real API or backend later.

## Stack

- React 19 + TypeScript
- Vite 7
- React Router 7
- Tailwind CSS 4
